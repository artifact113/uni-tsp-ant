Zum öffnen und verwenden der Projekte wird muss das TAO-Framework installiert werden.
Dazu die in diesem Ordner beiliegende "Tao-1.2.0-2-win32-Setup.exe" installieren.

Nach Abschluss der Installation, müssen ggf. die DLLs in die Projekte eingefügt werden.
Dazu im Projektmappen-Explorer auf den Eintrag "Verweise" rechtsklicken->"Verweis hinzufügen..". 
Mit dem Reiter "Durchsuchen" die folgenden DLLs hinzufügen:
	Tao.OpenGl.dll
	Tao.Plattform.Windows
